# ePortfolio

Hello any and all viewers. This is my (Gaidge Jellick) portfolio on the majority of what I learned in college while going for my Bachelor's Degree in Computer Science.

It covers a specific set of enhancements I made to one of my previous final projects on a weight-tracking application. These enhancements not only show off my skills and knowledge from what college has taught me, but also how my brain thinks as a computer science major. 

Within this space, you will find 2 Major Files. 

The first is the Old project file. This runs the un-enhanced version of my software.

The second is my fully enhanced and updated software, which includes all enhancements starting from the first all the way to the third. Since I did my enhancements all on the same software rather than different ones, you will see the same software when looking at each instance of it. However, each iteration tackles some key concepts and issues that I had with the software. Each enhancement is also paired with its written overview of changes I made and why.
